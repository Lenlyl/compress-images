# 批量处理图片压缩至指定大小

#### maxSizeKB 指定图片最大大小，单位为KB
[github源码 >>](https://github.com/Lenlyl/compress-images.git)

## 使用步骤
### node版本要求>=18.17.0 
[npm sharp](https://www.npmjs.com/package/sharp) <br>
[sharp文档](https://sharp.pixelplumbing.com/)

```bash

# install dependency
npm install

# develop
node images.js

```
